== Ten Pin Bowling

This is the first attempt to build a small Rails application to calculate the total score of a ten pin bowling game with given inputs.

The calculation of the scores needs some fine-tuning for greater accuracy.

=== Ruby version

2.1

=== Rails version

4.0.4

=== How to run the test suite

bundle exec rspec OR bundle exec rake

=== How to run the Rails application

rails server ( default: localhost:3000 )
